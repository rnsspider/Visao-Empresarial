# Instruções do OpenCode

- **Rodar a aplicação**: `npx http-server` (ou outra ferramenta de servidor estático).
- **Build**: não há build; modificar arquivos dentro de `js`, `css` ou os html.
- **Commit**: adicione as alterações e faça commit normalmente.
- **Teste**: não há testes automatizados. Qualquer verificação manual deve ser feita no navegador.
- **Distribuição**: não há pipeline; publicar os arquivos estáticos.

Todos os arquivos importantes estão na raiz; não há sub‑repos.
